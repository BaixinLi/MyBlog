# Baixin-Blog

欢迎来到 **Baixin-Blog**，这是我的个人知识笔记博客。我在这里记录我的读书笔记、课程心得和生活感悟。这个博客既是我分享探索世界的平台，也是我整理和沉淀个人思考的空间。

## 关于这个项目
这是一个基于Jekyll构建的静态网站，托管在GitHub Pages上。我的目标是通过每月更新Markdown文件，持续记录和分享我的学习与生活。

## 主要内容
- **读书笔记**：我读过的书以及从中获得的启发。
- **课程心得**：学习课程时的关键收获和总结。
- **生活感悟**：日常观察和对生活的点滴思考。
